User's guide for the Extensible Human Oracle Suite of Tools
(Stand-alone client Alpha version)

This guide provides information for users of the extensible Human Oracle Suite of Tools (eHOST) application. Multiple features were integrated with the eHOST tool with the intention of improving annotation efficiencies and reducing annotator workload. This guide will focuses on the annotation features of eHOST. It describes what eHOST is, how to get started using it, how to annotate documents with it, and what the basic workflow is for annotation tasks. Furthermore, since eHOST is a Java application which is capable of running on many different platforms, this guide utilizes Windows XP SP3 as its platform for documentation purposes.
