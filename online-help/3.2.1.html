3.2.1 Generating Graph Reports of Position Indicated

1. In the viewer Panel, Click the third tagged Label called "Reports"

2. Click the button called Graph reports of Position Indicated.

For the output graph, it indicates the annotations’ position by each of the file and the length of each annotation. It will also show with multiply files. The height of each box means the length of each file. Each colored line indicated the position and the spam of the annotations in that file.
