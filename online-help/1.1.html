1 Installation and Quick Start

1.1 Prerequisites

  • eHOST can run on any platform that runs Java including:
    • Windows XP, Vista, and 7
    • Apple MacOS X
    • Solaris
    • Linux
  • Must have Java installed for your platform:
    • For Windows, Solaris, and Linux, get the latest version from Java.com (http://www.java.com/en/download)
    • For MacOS X, run Software Update (http://support.apple.com/kb/HT1338?viewlocale=en_US)
