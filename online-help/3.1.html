3 Advanced Features

3.1 Annotation Review and Curation

1. Click Markables

2. Select "Person" to see the only the Person class is selected.

3. The right bar shows the position for "Person" in current document.

4. Click in the pane to expand and folder the list of class "Person".

5. Double Click "Her" to check the annotation.

6. Right Click the selected sub class "Her" to delete the current annotation.
